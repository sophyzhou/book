Data Models

The data model for this project would require 3 types of data.

  • Users
  • Movies
  • Chatting Rooms

To ackomplish this We could create a data model like below:

  • Users

    • key - user
      • key - username
      • key - email
      • key - imageURL

  • Movies

    • key - movie
      • key - title
      • key - YouTubeID
      • key - associatedChatroom

  • Chatrooms

    • key - chatroom
      • key - usersinthisRoom
        • key - user1
          • key - username
          • key - imageURL
        • key - user2
          • key - username
          • key - imageURL
      • key - record
        • key - record1
          • key - username
          • key - text
        • key - record2
          • key - username
          • key - text
